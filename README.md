## Description

[CryptoPortfolioAPI](https://github.com/gusilva/crypto-server) Crypto portfolio API.

## Installation

```bash
$ npm install
```

## Configuration

Copy the env example contents to the `.env` file
```bash
$ cp .env-example .env
```
Alter the `MONGO_URL` to your desired mongo db host

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
